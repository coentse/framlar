          
#### 基于Laravel-5.6的框架

- 版本
> php-7.1.8; laravel-5.6; composer-1.6.3

- 搭建
    - composer install
    - npm install

- 使用
    - /api 接口
    - /manage 后端
    - /callback 回调
    - /wechat 微信回调
    - /open 微信开放平台回调
    - /serve 消息处理

- 接口文档
    > npm install apidoc -g 全局安装apidoc

    - http://apidocjs.com/#params apidoc官方文档
    - /api/manage/doc 后端接口文档查看
    - /api/client/doc 客户端端接口文档查看


.env 文件
APP_KEY生成  php artisan key:generate
